### **Bài 8**
*   Tổng quãng đường s = 400 + 300 = 700 m.
*   Tổng thời gian t = 360 + 240 = 600 s.
*   Độ dời d = √(400² + 300²) = 500 m.
*   **Tốc độ trung bình:** v_tb = s / t = 700 / 600 ≈ **1,17 m/s**.
*   **Vận tốc trung bình:** v = d / t = 500 / 600 ≈ **0,83 m/s**.

### **Bài 9**

*   **Tốc độ trung bình:**

    *   t = 1 năm ≈ 3,156×10⁷ s.
    *   v_tb = s / t ≈ **29845 m/s** (≈ 29,85 km/s).
